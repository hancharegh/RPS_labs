# Лабораторная работа №1
## Тема
Изучение основ низкоуровневого программирования на примере Машины Тьюринга.

## Вариант 1
**Задание:** Удалить все символы `a`, оставив только `b`.

**Пример:**
```
Вход: aababb  
Выход: bbbb
```

## Описание алгоритма
1. Считать текущий символ.
2. Если `a` — заменить на `_` и сдвинуться вправо.
3. Если `b` — перейти к следующей ячейке вправо.
4. Когда встречается пустая ячейка `_` — завершить работу.

## Таблица переходов
| Состояние | Вход | Запись | Сдвиг | Следующее |
|------------|------|--------|--------|-------------|
| q0 | a | _ | R | q0 |
| q0 | b | b | R | q0 |
| q0 | _ | _ | N | qf |

## Тестирование
| Вход | Выход | Комментарий |
|------|--------|--------------|
| aababb | bbbb | удалены все `a` |
| aaaa | *(пусто)* | все символы удалены |
| bbbab | bbbb | удалены все `a` |
